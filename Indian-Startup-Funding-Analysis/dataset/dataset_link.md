# Dataset Information

## Dataset Name

Indian Startup Funding Dataset (2015–2019)

## Source

Kaggle

## Original Dataset

https://www.kaggle.com/datasets/sudalairajkumar/indian-startup-funding

## Description

This dataset contains funding records of Indian startups between 2015 and 2019, including startup name, industry, city, investors, funding amount, investment type, and funding date.

## Notes

- Imported into Google BigQuery for SQL analysis.
- The original dataset was not modified before import.
- Data cleaning and standardization were performed within SQL queries during analysis.
