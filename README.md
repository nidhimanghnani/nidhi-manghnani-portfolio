# Projects

A collection of hands-on projects demonstrating structured problem-solving, analytical thinking, product strategy, Agile execution, and data-driven decision-making.

## [Project 1: Indian Startup Funding Analysis](./Indian-Startup-Funding-Analysis)

Analysis of real-world startup funding data to identify investment trends, sector-wise funding patterns, and business insights using SQL and Google BigQuery.

**Skills:** Data Analysis, SQL, Google BigQuery, Business Insights

---

## [Project 2: CRED Product Teardown & PRD](./CRED-Teardown-and-PRD)

A product teardown of CRED's Referral Hub, including feature analysis, problem identification, user stories, prioritization and a Product Requirements Document (PRD).

**Skills:** Product Teardown, Product Requirements Document (PRD), User Stories, Prioritization, Feature Scoping

---

## [Project 3: CRED Referral Hub – Agile Sprint Tracker](./CRED-Referral-Hub-Sprint-Tracker)

A simulated Agile sprint built from the user stories and PRD created in Project 2, covering backlog grooming, sprint planning, Kanban tracking, daily standups, burndown tracking, and sprint retrospectives.

**Skills:** Agile, Scrum, Sprint Planning, Backlog Management, Story Point Estimation, Kanban, Sprint Retrospectives, Notion
